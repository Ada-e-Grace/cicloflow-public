Política de Privacidade

English

Resumo rápido

  • Seus dados ficam apenas neste aparelho.
  • Você escolhe o que registrar e pode apagar tudo quando quiser.
  • O app não faz diagnóstico nem envia suas informações para servidores.

1. Quem somos

O CicloFlow é um aplicativo voltado para educação em saúde menstrual e acompanhamento do ciclo, pensado para ajudar você a entender melhor suas fases, sintomas e seu bem-estar ao longo do mês. Esta Política de Privacidade explica, em linguagem simples, como seus dados são tratados dentro do aplicativo.

2. Quais dados tratamos

Para funcionar, o CicloFlow utiliza apenas os dados necessários para calcular e acompanhar o seu ciclo menstrual. Em especial:

  • Dados de identificação básicos: nome ou apelido que você escolhe informar.
  • Dados do ciclo: data da última menstruação, duração média do ciclo, duração média da menstruação.
  • Registros de sintomas e humor: quando você opta por registrar como se sente e quais sintomas aparecem ao longo do ciclo.
  • Preferências locais: configurações de notificações, lembretes e avisos dentro do próprio app.

Essas informações sobre ciclo, sintomas e humor são consideradas dados pessoais sensíveis pela LGPD, pois se relacionam à sua saúde.

3. Onde e como seus dados são armazenados

Todos os dados que você registra no CicloFlow ficam salvos apenas neste aparelho. O aplicativo utiliza um banco de dados local e preferências locais do sistema para guardar suas informações.

Isso significa que:

  • Não enviamos seus dados para servidores externos ou para a nuvem.
  • Não fazemos cópia automática dos seus dados em serviços de terceiros.
  • Não utilizamos seus dados para publicidade ou venda de informações.

4. Como usamos seus dados

Os dados que você registra são usados exclusivamente para:

  • Calcular e exibir as fases do seu ciclo menstrual em formato visual.
  • Ajudar a identificar padrões de sintomas e variações de humor ao longo do ciclo.
  • Oferecer conteúdos educativos e sugestões de autocuidado adequadas a cada fase.
  • Agendar notificações locais opcionais (como lembretes diários, avisos de fase lútea ou TDPM em curso), caso você escolha ativá-las.

Não utilizamos esses dados para criar perfis comerciais, enviar publicidade ou compartilhar suas informações com terceiros.

5. Base legal para o tratamento de dados (LGPD)

Como o CicloFlow lida com dados pessoais sensíveis (relacionados à sua saúde), a principal base legal utilizada é o seu consentimento, de forma livre, informada e inequívoca, conforme a Lei Geral de Proteção de Dados (LGPD).

Ao prosseguir no cadastro e usar o aplicativo, você concorda que o CicloFlow trate os dados informados para as finalidades descritas nesta Política de Privacidade e na tela de consentimento durante o cadastro.

6. Compartilhamento de dados

O CicloFlow não compartilha seus dados pessoais com terceiros.

Não enviamos seus dados para serviços de análise externos, anúncios, redes sociais ou parceiros comerciais. Tudo o que você registra é mantido apenas neste dispositivo para uso dentro do próprio app.

7. Por quanto tempo mantemos seus dados

Seus dados permanecem armazenados neste aparelho enquanto o aplicativo estiver instalado e enquanto você não optar por apagá-los.

Você pode, a qualquer momento:

  • Usar a opção "Excluir todos os meus dados deste aparelho" dentro das Configurações / Conta; e/ou
  • Desinstalar o aplicativo, o que também remove o banco de dados local deste dispositivo.

Após a exclusão, os dados não podem ser recuperados pelo aplicativo.

8. Seus direitos como titular de dados

De acordo com a LGPD, você tem direito a:

  • Acessar as informações que registrou no app.
  • Corrigir dados que estejam incorretos ou desatualizados (por exemplo, editar suas informações de ciclo).
  • Revogar o consentimento para tratamento dos dados, apagando suas informações pelo app ou desinstalando o CicloFlow deste aparelho.
  • Solicitar mais esclarecimentos sobre como seus dados são tratados.

No CicloFlow, muitos desses direitos podem ser exercidos diretamente nas telas do aplicativo, especialmente em "Configurações / Conta" e nos formulários de edição de dados.

9. Revogação de consentimento e exclusão de dados

Você pode revogar seu consentimento e interromper o uso dos seus dados sensíveis a qualquer momento, de duas maneiras:

  • Dentro do app: acessando "Configurações / Conta" e usando a opção "Excluir todos os meus dados deste aparelho".
  • Fora do app: desinstalando o CicloFlow do seu dispositivo.

Em ambos os casos, os dados armazenados localmente deixam de ser acessíveis pelo aplicativo neste aparelho.

10. Atualizações desta Política

Podemos atualizar esta Política de Privacidade de tempos em tempos, por exemplo, para refletir melhorias no aplicativo ou mudanças na legislação. Quando isso acontecer, a versão mais recente ficará disponível nesta tela, com a data da última atualização.

Recomendamos que você revise este conteúdo periodicamente para acompanhar como seus dados são tratados.
